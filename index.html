Development


The Ship Supplier

X

I was tasked with rebuilding the whole website for The Ship Supplier from the ground up. The old site had become dated with broken components and needed a full refresh. The design had to be kept simple and user friendly and aimed at a corporate audience. I also had to keep a few things in mind when designing the UI. The colour scheme needed to be simple to match the branding colour scheme, a simple UI almost like a news site with some extra compents to also modernise it and also to add some to functionality for usability. This included making it mobile responsive, adding a contact form, connecting it to social media channels and adding a digital flipbook for the digital magazines all of which didn't exist in the previouse website

The final outcome both fixes all of these issues as well as giving the website a new clean aesthetic adding new elements like home page videos and transition in image galleries to help set the design off to prevent it looking plain.

VISIT THE LIVE WEBSITE WWW.THESHIPSUPPLIER.COM

SOFTWARE: HTML | CSS | JAVASCRIPT | VISUAL STUDIO CODE | PHOTOSHOP | AFTER EFFECTS
